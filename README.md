# Windek Group Website

This project is a static corporate website built for Windek Group.

It presents the company's brand, subsidiaries, leadership team, investor positioning, contact details, and supporting business information in a modern responsive layout.

## Project Overview

The website was designed as a modern single-page company profile for Windek Group with a focus on:

- brand presentation
- subsidiary visibility
- leadership presentation
- investor-facing credibility
- mobile responsiveness
- animated user interface sections

## Features

- modern hero section with animated visual layout
- subsidiaries section with direct links to company websites
- subsidiary operations section
- projects and industries sections
- investor relations section
- leadership team section with image-based cards
- contact section with clickable phone and email links
- Google Maps office embed
- responsive layout for desktop, tablet, and mobile

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## How To Run

1. Open the project folder.
2. Open `index.html` directly in a browser.

You can also run it with any simple local server if preferred.

## Folder Structure

- `index.html`  
  Main website structure and content.

- `style.css`  
  All layout, design, animation, and responsive styles.

- `script.js`  
  Interactive behavior such as navigation, animations, tabs, counters, modal actions, and form validation.

- `img/`  
  Logo files and image assets used by the website.

- `README.md`  
  Project documentation.

## Customization

You can easily update the following:

- company logo in `img/`
- contact details in `index.html`
- team member names, roles, and images in the team section
- subsidiary links in the subsidiaries section
- social media links in the footer
- colors and spacing in `style.css`

## Current Limitations

- some team images are placeholder SVG portraits
- some social media links may still use placeholder `#` links unless replaced
- no backend or live form submission is connected
- content is static and must be edited manually
